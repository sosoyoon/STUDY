### 1. 현행시스템 분석
1) 현행 시스템 파악<br>
(1) 현행시스템 파악의 정의
   - 현행 시스템이 어떤 하위 시스템으로 구성되어 있는지
   - 제공하는 기능이 무엇인지
   - 다른 시스템들과 어떤 정보를 주고받는지 ➡️ 인터페이스
   - 어떤 기술요소를 사용하고 있는지
   - 사용하고 있는 소프트웨어 및 하드웨어는 무엇인지
   - 네트워크는 어떻게 구성되어 있는지

2) 플랫폼 기능 분석<br>
